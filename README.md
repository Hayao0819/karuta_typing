# Jomo Karta Typing App
---
## Lets typing !

- 開発環境での起動方法(win、mac共通)
  - `git clone https://github.com/yohei-koba-vitalize/karuta_typing.git`
  - `git pull origin master`
  - `npm install`
  - 環境変数ファイル ".env" を以下の通り ”/karuta_typing/" ディレクトリ配下に格納する
  - `npm run dev`
